# Countries Search

React + Vite app that displays country flags and names and supports search.

Run locally:

1. Install dependencies:

```bash
npm install
```

2. Start dev server:

```bash
npm run dev
```

3. Open the URL shown by Vite (usually `http://localhost:5173`).

Notes:
- Search input is an `<input type="text">`.
- Each country card uses the class `countryCard` (required by tests).
- Flags are rendered using `<img>` elements.
- Network errors are logged to the console.
